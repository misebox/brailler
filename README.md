# brailler

This tool converts images into Braille dot pattern text.

## Usage

```
$ brailler --help
Usage: brailler [OPTIONS] <INPUT>

Arguments:
  <INPUT>  Input image file path or video file path

Options:
  -s, --size <SIZE>            Output size({width}x{height}) [default: 0x0]
      --contrast <CONTRAST>    Contrast option [default: none] [possible values: none, stretch, equalize]
      --invert                 Invert dot color
      --binarize <BINARIZE>    Binarize option [default: none] [possible values: none, odith, fsdith, otsu]
      --scriptify <SCRIPTIFY>  Generate a bash script that shows the braille text [default: ]
  -v, --verbose                Verbose mode
  -h, --help                   Print help
  -V, --version                Print version
```

## size

`ex: brailler samples/Vermeer_The_Girl_With_The_Pearl_Earring_B.jpg --size {cols or 0}x{rows or 0}`

This option allows you to specify the size of the output.
Sizes are specified in the form of "{cols}x{rows}".
Specify two numbers separated by "," or "x".
If you specify zero for one side, the size will be based on the image ratio.
If not specified or specified 0x0, it will behave the same as if 0x40 was specified.

## Preprocess

### contrast

- `--contrast stretch` option allows you to preprocess the image with Contrast Stretch.
- `--contrast equalize` option allows you to preprocess the image with Histogram Equalization.

### invert

- `--invert` option allows you to invert the image.


```
paste -d' '  \
  <( brailler samples/Vermeer_The_Girl_With_The_Pearl_Earring_B.jpg --size 50x0 ) \
  <( brailler samples/Vermeer_The_Girl_With_The_Pearl_Earring_B.jpg --size 50x0 --contrast stretch ) \
```

### Binarize

- `--binarize odith` option allows you to binarize the image with Ordered Dithering.
- `--binarize fsdith` option allows you to binarize the image with Floyd-Steinberg Dithering.
- `--binarize otsu` option allows you to binarize the image with Otsu's method.

```
  <( brailler samples/cube.png --size 50x0 --binarize otsu )
```

## Video

- `--video` option allows you to convert video to Braille dot pattern text.

```
# Convert video to Braille dot pattern text
$ brailler samples/test.mp4 --contrast stretch --binarize fsdith --size 0x60
```

## Scriptify

- `--scriptify` option allows you to generate a shell script to show the braille text.

```
$ brailler samples/test.mp4 --contrast stretch --binarize fsdith --size 0x60 --scriptify play_mov.sh

$ ./play_mov.sh
```
